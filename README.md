# Code-Along: To-Do List Application

## Description
This repository contains the code developed during a software engineering code-along session. The project focuses on building a functional To-Do List application using Object-Oriented JavaScript, specifically focusing on constructors and prototypes.

## Learning Objectives
* Implementing JavaScript constructors and prototypes.
* Managing state within an object-oriented structure.
* Manipulating the DOM to reflect data changes.
* Handling form submission and click events in JavaScript.

## Features
* Add new tasks with descriptions.
* Assign unique IDs to tasks automatically.
* Delete tasks from the list.
* Persistent logic using a global ToDoList instance.
